Raoul DIFFOUO Blog
==================

My jekyll powered blog  
Visit blog: http://blog.diraulo.me/

### Features
- Pagination. (New)
- Syntax highlighting with pygments. (New)
- Responsive layout.
- Supports tags and categories.
- Social profile and bio of author.
- Bootstrap based.
- Glyphicon and Font-Awesome Icons.
